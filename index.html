Perguntas frequentes

Preciso instalar alguma coisa?

Não. A ferramenta roda inteiramente no navegador, sem plugins ou aplicativos extras.

Minhas imagens são enviadas para algum servidor?

Não. Todo o processamento acontece localmente no seu dispositivo. Nenhuma imagem é transmitida. Leia mais na Política de Privacidade.

Posso carregar mais de uma imagem?

Sim. Selecione múltiplos arquivos de uma vez, arraste vários, ou carregue imagens separadamente — cada uma ganha seu próprio painel de extração.

Posso usar no celular?

Sim. Toque e arraste para visualizar a cor em tempo real. Solte para selecioná-la.

O que é a paleta dominante?

Ao carregar uma imagem, a ferramenta analisa os pixels e apresenta as cores mais frequentes, facilitando a criação de paletas para design.

HEX, RGBA e HSL — qual formato usar?

HEX é o formato mais usado em CSS e ferramentas de design. RGBA é ideal quando você precisa controlar a transparência. HSL facilita ajustes de tonalidade, saturação e luminosidade, tornando a geração de variações de cor muito mais intuitiva.
